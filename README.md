# Odd-Even — Rigged Dice Game (Gambling Awareness Demo)

A single-page, self-contained gambling-awareness demo built for classroom/booth use. Players guess odd/even on two dice; the game secretly manipulates win odds to teach how rigged gambling apps can look fair while quietly stacking the odds against the player.

Open `index.html` directly in a browser, or visit the GitHub Pages deployment once enabled.

No backend, no build step, no dependencies beyond Google Fonts.
